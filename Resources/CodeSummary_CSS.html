Backgrounds

Try not to confuse background images with positioned images. Background images are in the container's Background. All of the rest of the content lives on top of it.

  • For content to scroll under a background image, put the image in a container, then make the container fixed and give it a z-index

Background Properties

  • background-color:
    • Values may be hex, rgb, rgba, or approved color names
  • background-image: url(whateverImage.jpg);
    • No need for quotes
    • Support for GIF, JPG, PNG, SVG
  • background-repeat:
    • Values may be repeat, no-repeat, repeat-x (to repeat the image horizontally, or in a row), repeat-y (to repeat values vertically, or in a column)
  • background-position: [horizontal] [vertical]
    • Default is top center of its containing element
    • Takes 2 numeric values:
      The first value is its position from the left edge
      The second value is its position from the top edge
    • Also may use keywords top, bottom, right, left, center
  • background-attachment:
    • fixed: fixes the background in the viewport - prevents it from scrolling away
      • If you have a relatively placed container with a fixed background, the container will scroll up and down the screen and the background will be fixed, so the container will appear to scroll through it, not overlapping other fixed containers, because the containers are relatively placed, thus fixing the background images in different places
      • To be sure text may scroll under (or over) the image, be sure to use a z-index on the container
    • scroll: the image scrolls along with the rest of the page
  • background-size: width [height]
    • May use a percentage, or pixel width
      • A percentage will be relative to the containing element, or the viewport if the attachment is fixed
      • 100% will make the image the width of the container, and however high it is without stretching
    • cover - scales the image to cover the entire container, so part of the image may be clipped to preserve the aspect ratio of the image
    • contain - letterboxes the image to preserve the aspect ratio, in order to fill either the width or height of the container
